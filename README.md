# Voice-Based-Todo-List

This is a Todo List application where you use voice commands to simply add or remove items from a Todo List. The voice command is powered by Alan AI, which is a powerful conversational AI. User can dynamically add items using voice commands allowing a new way to use a Todo List. 

Check out how it looks: https://sakibur-karim.github.io/Voiced-Todo-List/

## How to install:

- Clone git repository
- Install the dependencies:<br />
<pre>npm i</pre>
- Start the server:<br />
<pre>npm start</pre>
